# Описание интерфейса и  возможностей

**ByeDPI Manager** представляет из себя оболочку для запуска совместной работы двух приложений: собственно **ByeDPI** и **ProxiFyre**.

**ByeDPI** - SOCKS прокси-сервер, который изменяет запросы, проходящие через него. Подробнее на [странице проекта на GitHub](https://github.com/hufrea/byedpi).

**ProxiFyre** - проксификатор - инструмент, который заставляет приложения работать через прокси-сервер. В данном случае через ByeDPI. Более подробно описано на [странице проекта на GitHub](https://github.com/wiresock/proxifyre).

Таким образом, стандартная работа **ByeDPI Manager** заключается в том, чтобы при запуске данных компонентов "захватывать" запросы от указаных приложении с помощью **ProxiFyre** и направлять его на локальный прокси-сервер **ByeDPI**.

Далее будут рассмотрены возможности настроек данных инструментов через **ByeDPI Manager** и не только.
